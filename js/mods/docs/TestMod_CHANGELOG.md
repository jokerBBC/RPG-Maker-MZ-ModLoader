# TestMod 更新日志

## V1.2.0 (2026-05-15)

### 边界测试增强
- **新增**：添加边界测试用例，验证帮助文本中的 `@` 符号不会被误解析
- **新增**：测试 `@help` 块内的 `@version`、`@author`、`@param` 等文本作为纯文本显示
- **优化**：调试日志输出分类整理，更清晰易读

## V1.1.0

### 参数类型扩展
- **新增**：支持数据库引用类型（actor/skill/item/weapon/armor/enemy/state）
- **新增**：支持长文本类型（note/multiline_string）
- **新增**：支持颜色类型（color）

### 功能优化
- **优化**：每日签到奖励银币功能
- **优化**：文字显示效果增强

## V1.0.0

### 初始版本
- **新增**：全能参数测试仪，验证 ModLoader 的参数编辑功能
- **支持**：数值、开关、选择、文本类型参数
- **新增**：每日签到发银币功能
- **新增**：调试日志输出功能
